# 🌐 Personal Portfolio Website

A modern, responsive, and interactive personal portfolio website developed using **React**, **TypeScript**, and **Vite**. This portfolio showcases my education, technical skills, projects, experience, and contact information in a clean and user-friendly interface.

---

## 📌 Features

- 👤 Professional Profile Section
- 🎓 Education Details
- 💻 Technical Skills
- 🚀 Projects Showcase
- 💼 Experience Section
- 📞 Contact Information
- 📱 Fully Responsive Design
- ⚡ Fast Performance with Vite
- 🎨 Modern UI Design

---

## 🛠️ Tech Stack

## 🚀 Installation

### 1. Clone the repository

```bash
git clone https://github.com/yourusername/portfolio.git
```

### 2. Navigate to the project folder

```bash
cd portfolio
```

### 3. Install dependencies

```bash
npm install
```

### 4. Start the development server

```bash
npm run dev
```

Open your browser and visit:

```
http://localhost:3000
```

---

## 📦 Build for Production

```bash
npm run build
```

The production files will be generated inside the **dist** folder.
